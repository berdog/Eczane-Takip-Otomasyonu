# Eczane-Takip-Otomasyonu
C# Windows Forms ve SQL Server kullanılarak geliştirilmiş rol bazlı eczane takip ve satış otomasyonu.

# 💊 Eczane Takip Sistemi

Bu proje C# Windows Forms ve SQL Server kullanılarak geliştirilmiş bir eczane otomasyon sistemidir.

## 🚀 Kullanılan Teknolojiler
- C#
- Windows Forms
- SQL Server
- ADO.NET

## 📌 Proje Özellikleri
- Kullanıcı giriş sistemi
- Rol bazlı yetkilendirme
- Admin paneli
- İlaç ekleme / silme / güncelleme
- Stok takibi
- Satış sistemi
- Günlük kazanç hesaplama
- Son kullanma tarihi kontrolü
- Kategori filtreleme

## 🗄 Veritabanı Kurulumu
Projeyi çalıştırmadan önce `EczaneTakipDB.sql` dosyasını SQL Server üzerinde çalıştırınız.
